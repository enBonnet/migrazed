## Token Colors

Token colors define syntax highlighting for various code elements. Each token has a scope and settings including foreground color and font style.

### Text Styles

- `emphasis` - Font style: italic
- `strong` - Font style: bold
- `header` - Foreground color for headers
- `markup.underline` - Font style: underline
- `markup.bold` - Font style: bold
- `markup.heading` - Font style: bold
- `markup.italic` - Font style: italic

### Comments

- `comment` - Comments
- `punctuation.definition.comment` - Comment punctuation
- `comment keyword.codetag.notation` - JSDoc-style keywords
- `comment.block.documentation keyword` - JSDoc keywords
- `comment.block.documentation storage.type.class` - JSDoc class storage

### Constants and Literals

- `constant` - Constants
- `variable.other.constant` - Constant variables
- `constant.character.escape` - Escape sequences
- `constant.character.string.escape` - String escape sequences
- `constant.regexp` - Regular expression constants
- `constant.other.date` - Date constants
- `constant.other.timestamp` - Timestamp constants
- `constant.other.symbol.hashkey.ruby` - Ruby symbol hash keys

### Entities

- `entity.name.tag` - HTML/XML tags
- `entity.other.attribute-name` - Attribute names
- `entity.other.attribute-name.parent-selector` - CSS parent selectors
- `entity.name.function` - Function names
- `entity.name.type` - Type names
- `entity.name.type.class` - Class names
- `entity.name.class` - Class names
- `entity.other.inherited-class` - Inherited classes
- `entity.name.type.type-parameter` - Generic type parameters
- `entity.name.fragment.graphql` - GraphQL fragment names
- `entity.name.filename` - File names
- `entity.name.function.target.makefile` - Makefile targets
- `entity.name.section.toml` - TOML sections
- `entity.name.tag.yaml` - YAML tags
- `entity.name.directive.restructuredtext` - reStructuredText directives

### Keywords

- `keyword` - Keywords
- `punctuation.definition.keyword` - Keyword punctuation
- `keyword.control.new` - "new" keyword
- `keyword.operator.new` - "new" operator
- `keyword.expressions-and-types.swift` - Swift expressions and types

- `keyword.type.cs` - C# types

## Usage Example

```json
{
  "$schema": "vscode://schemas/color-theme",
  "name": "My Theme",
  "author": "Your Name",
  "colors": {
    "editor.background": "#191917",
    "editor.foreground": "#f3f3f3",
    "terminal.background": "#191917",
    "terminal.foreground": "#f3f3f3",
    "activityBar.background": "#20201e",
    "sideBar.background": "#151413"
  },
  "tokenColors": [
    {
      "scope": ["comment"],
      "settings": {
        "foreground": "#e95421"
      }
    },
    {
      "scope": ["keyword"],
      "settings": {
        "foreground": "#e858bc"
      }
    }
  ]
}
```
